<h1>CyberCafe Management System</h1>

A **web-based CyberCafe Management System** developed using **PHP** and **MySQL** to manage users, computer bookings, billing efficiently. 
This system is designed to simplify the management of cyber cafes by providing an intuitive interface for both admins and users.

---

<h3>Project Overview</h3>

This system has two types of users: **Admin** and **User**, each with distinct roles and permissions.

---

## Roles and Permissions

### Admin

The admin is responsible for managing the overall operations of the cyber cafe. Admin can:

-check users.
- Create and manage user bills
- Manage computer availability and bookings.
- add delete update computers

### User

A user can:

- Log in and view available computers.
- use computers according to available packages.
- View their usage history and billing information.

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server Environment:** XAMPP (Apache + MySQL)

---

## Requirements

Before running this project, ensure you have:

- **XAMPP** installed on your system  
- Access to **phpMyAdmin** to manage the MySQL database  
- A modern web browser (Chrome, Firefox, etc.)
